# Происхождение данных

## data/input/ — подвыборка LEVIR-CD (val, папка A «before»)

- Датасет: LEVIR-CD — 637 пар VHR-снимков Google Earth 1024x1024 (0.5 м/пиксель),
  бинарная разметка изменений застройки. Официальная страница:
  https://justchenhao.github.io/LEVIR/
- Источник загрузки: зеркало Hugging Face
  `satellite-image-deep-learning/LEVIR-CD`, файл `val.zip` (папки A/B/label,
  по 64 снимка).
- Взяты 10 снимков «до изменений» (val/A), равномерно по набору
  (разные территории): val_1, val_7, val_13, val_19, val_25, val_31,
  val_37, val_43, val_49, val_55 — с префиксом `levir_`.
- Назначение: исходники для сезонной генерации (лето/осень/зима/глубокая зима).

## Лицензия

Изображения и аннотации LEVIR-CD разрешены **только для академического
использования**, коммерческое использование запрещено. Также действуют условия
использования Google Earth. Файлы датасета не коммитятся в репозиторий
(см. .gitignore), скачиваются скриптом.

## Повторная загрузка

```bash
python scripts/download_levir.py
```
